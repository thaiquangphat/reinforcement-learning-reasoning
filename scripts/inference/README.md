# Inference Scripts

This directory contains inference scripts for running trained reinforcement learning models:

- **inference_rl.py** - Base inference script
- **inference_rl_0.py** - Variant 0 inference
- **inference_rl_0_1.py through inference_rl_0_5.py** - Variant 0 sub-experiments
- **inference_rl_2.py** - Variant 2 inference
- **inference_rl_3.py** - Variant 3 inference

Each script runs inference using the corresponding trained model variant.
Models are loaded from the checkpoints_archive directory.
