# Training Scripts

This directory contains training scripts for reinforcement learning models:

- **train_rl.py** - Base training script
- **train_rl_0.py** - Variant 0 training
- **train_rl_0_1.py through train_rl_0_5.py** - Variant 0 sub-experiments
- **train_rl_2.py** - Variant 2 training
- **train_rl_3.py** - Variant 3 training

Each script trains different model configurations or uses different hyperparameters.
Checkpoints are saved to the checkpoints_archive directory.
